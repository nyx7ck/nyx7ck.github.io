Determine if the given function is a linear transformation.

$$T(\begin{bmatrix} x_1 \\ x_2 \end{bmatrix}) = \begin{bmatrix} 3x_1 + x_2 \\ -2x_1 + 4x_2 \end{bmatrix}$$







Assume $$\vec{u} = \begin{bmatrix} u_1\\ u_2 \end{bmatrix}, \quad \vec{v} = \begin{bmatrix} v_1\\ v_2 \end{bmatrix}$$


Let’s check vector addition:

$$T(\vec{u}) + T(\vec{v}) = \begin{bmatrix} 3u_1 + u_2 \\ -2u_1 + 4u_2 \end{bmatrix} + \begin{bmatrix} 3v_1 + v_2 \\ -2v_1 + 4v_2 \end{bmatrix}$$


$$T(\vec{u}) + T(\vec{v}) = \begin{bmatrix} 3(u_1 + v_1) + (u_2 + v_2) \\ -2(u_1 + v_1) + 4(u_1 + u_2) \end{bmatrix}$$

\[ T(\vec{u}) + T(\vec{v}) = T(\vec{u} + \vec{v}) \]


Vector addition is closed!



Let’s check the scalar multiplication:


$$rT(\vec{u}) = r\begin{bmatrix} 3u_1 + u_2 \\ -2u_1 + 4u_2 \end{bmatrix} = \begin{bmatrix} 3ru_1 + ru_2 \\ -2ru_1 + 4ru_2 \end{bmatrix} = T(r\vec{u})$$


Scalar Multiplication is closed



Thus, \(T(\vec{x})\) is a linear transformation.
