(a) Express the double integral $$\int_{0}^{2} \int_{x}^{3x-x^2} dy dx$$ as an iterated integral for the given function \(f\) and region \(D\).



(b) Evaluate the iterated integral.









(a). Notice this is the first type region, so we have $$\int_{0}^{2} \int_{x}^{3x-x^2} dy dx$$

(b). Now, we are going to solving this double integral by solving the inner integral firstly $$\int_{0}^{2} (3x-x^2-x) dx$$

Next, we solve the outer integral

$$\int_{0}^{2} (2x-x^2) dx$$

At last, we compute the result

$$x^2 - \frac{x^3}{3} \Big|_{0}^{2}$$

$$4 - \frac{8}{3}$$

$$\frac{4}{3}$$

Thus, the answer for this double integral is $$\frac{4}{3}$$
