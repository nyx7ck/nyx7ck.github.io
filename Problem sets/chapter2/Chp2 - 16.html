Suppose A is a matrix with n rows and m columns. If n < m, then the columns of A span Rn







Strategy: if the matrix looks like the shape below (vertical chunk):

$$\begin{bmatrix} * & * \\ * & * \\ * & * \end{bmatrix}$$

it must not be able to span Rn.

However, the questions was asking for the horizontal chunk like this:

$$\begin{bmatrix} * & * & * \\ * & * & * \end{bmatrix}$$

So this type of matrix could be either span or not span, if we can make an counterexample, then we could verify the statement is false.

Here is an counterexample: $$\begin{bmatrix} 1 & 0 & 0 \\ 0 & 0 & 0 \end{bmatrix}$$ This matrix has three column vectors, but couldn’t span \(R^2\)

Thus, the statement is False.
