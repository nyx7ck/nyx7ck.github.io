If \(\lambda_1\) and \(\lambda_2\) are eigenvalues of a matrix, then so is \(\lambda_1 + \lambda_2\)






We proceed by contradiction

Assume both eigenvalues are positive:

\[ \lambda_3 = \lambda_1 + \lambda_2 \]
\[ \lambda_4 = \lambda_1 + \lambda_3 \]
\[ \lambda_5 = \lambda_3 + \lambda_4 \]
Notice we can produce infinite eigenvalues because addition is transitive.

If a matrix is \(3 \times 3\) , then we can produce infinite eigenvalues. Obviously it is impossible.

Thus, the statement is False
